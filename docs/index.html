PDF LIVE

'PDF LIVE' is a PDF viewer that extends the open source 'mozilla/pdf.js'. You can start the PDF viewer with toolbar right away with 'PDF LIVE'. 

npm i pdf-live

Currently v1.0.0 • Install • ES6 Usage • GitHub repo

Install

Install with npm:

npm i pdf-live
